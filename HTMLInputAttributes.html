HTML Input Attributes

In this we will discuss the different attributes for the HTML <input> element.

The Value Attribute

The input value attribute specifies the initial value for an input field.

The Readonly Attribute

The input readonly attribute specifies that an input field is read-only.

The read-only input field cannot be modified (homepage, a user can tab to it, highlight it and copy the text from it).
The value of a read-only input field will be sent when submitting the form.

The size Attribute

The size attribute specifies the visible width, in characters of an input field.
The default value for size is 20.
Note: The size attribute works with the following input: text, search, tel, url, email and password.





The maxlength Attribute

The maxlength attributes specifies the maximum number of characters allowed in an input field.
Note: When a maxlength is set, the input field will not accept more than the specified number of characters.
However, this attribute does not provide any feedback. So if you want to alert the user, you must write JavaScript code.





The Pattern Attribute

The input pattern attribute specifies a regualar expression that the input-field's value is checked against
When the form is submitted.
The pattern attribute works with the following input types: text, date, search, url, tel, email and password.

The Placeholder Attribute

The input placeholdere attribute specifies a short hint that describes the expected value of an input field.
A sample value or a short Description of the expected format.
The short hint is displayed in the input field before the user enters a value.
The placeholder attribute workswith the following input types: text, seacrh, url, tel, email and password.


The required Attribute

The input required attribute specifies that an input field must be filled out before submitting the form.
The required attribute works with the follwing input types: text, search, url, tel, email, password, datepickers, number, checkbox
radio,file.

The Step Attribute

The step attribute specifies the legal number intervals for an input field.
Example: if step="3", legal numbers could be -3,0,3,6,etc.
Tip: This attribute can be used together with the max and min attributes to create a range of legal values.
If step attribute works with the following input types: number, range, date, datetime-local, month, time, and week.

The Height and Width Attributes

The input height and width attribute specify the height and width of an <input type="image"> element.

Tip: Always specify both the height and width attributes for images. If height and width are set, the space
required for the image is reserved when the page is loaded. Without these attributes, the browser does not know the size of the
image, and cannot reserve the appropriate space to it. The effect will be that the page layout will change during loading
(while the images load).
