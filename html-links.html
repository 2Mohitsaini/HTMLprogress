Link Target

By default the linked page will be displayed in the current browser window. To change this, you must
rectify another target for the link.
The target attribute specifies where to open the link.
The target attribute can have following values:
_self: Default. Open the document in the same window/tab as clicked.
_blank: opens the document in new window or tab. visit facebook
_parent: Opens the document in parent frame. parentlink
_top: Opens the document in the full body of the window. toplink

HTML Link Colors

By default a link will appear like this (in all browsers)
                    1. An unvisited link is underlined and blue
                    2. A visited link is underlined and purple
                    3. An active link is underlined and red.
