HTML Form Elements

This page will describe all the different HTML form elements.

The HTML <form> element can contain one or more of the following elemnents:

  • <input>
  • <label>
  • <select>
  • <textarea>
  • <button>
  • <fieldset>
  • <legend>
  • <datalist>
  • <output>
  • <option>
  • <optgroup>

The <input> Element

One of the most common form element is the input element.

The <input> element can be displayed in several ways, depeding on the type of attribute.

Some of them are already discussed, in next page will discuss more of them.

The<label> Element

The label element defines a label for several form elements

The label element is useful for screen-reader users, because the screen-readerwill read out loud the label when the user focus on the input element.

The label element also help users who have difficulty clicking on small regions (such as radio buttons or checkboxes) because when the user clicks the text within the labelelement, it toggles the radio button/checkbox.

The for attribute of the label tag should be equal to the id attribute of the input element to bind them together.

The <datalist> Element

The <datalist> element specifies a list of pre-defined options for an <input> element.

User will see a drop-down list of the pre-defined options as they input data.

The list attribute of the <input> element, must refer to the id attribute of the <datalist> element.
